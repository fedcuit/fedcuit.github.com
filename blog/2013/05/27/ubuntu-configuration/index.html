Setup ftp server

  • Install vsftpd sudo apt-get install vsftpd

  • Edit /etc/vsftpd.conf

    • anonymous_enable
    • write_enable
    • ftpd_banner

  • Save the file and restart ftp service by sudo /etc/init.d/vsftpd restart

  • Copy files you want to shared to folder /srv/ftp
